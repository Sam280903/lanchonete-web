# Lanchonete do Bairro

Site da Lanchonete do Bairro — projeto desenvolvido para a disciplina de Engenharia da Qualidade e Confiabilidade.

## Estrutura do Projeto

- `index.html` — Página principal com cardápio
- `promocoes.html` — Página de promoções do dia
- `style.css` — Estilos da aplicação
- `cardapio.js` — Lógica interativa do cardápio

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Navegue pelo cardápio e acesse as promoções pelo menu.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
